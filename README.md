# Aver

macOS用 Adobe アプリ バージョン確認ユーティリティ

## 概要

`/Applications` フォルダ内の Adobe アプリを自動スキャンし、バージョン情報を一覧表示します。バージョン文字列をクリップボードにコピーする機能付き。

## 機能

- Adobe アプリの自動スキャン・一覧表示（バージョン付き）
- macOS情報（Apple Silicon / Intel + OSバージョン・コードネーム）をヘッダーに表示
- アプリ名クリック → そのバージョン文字列をヘッダーに追加表示
- **コピーボタン**：ヘッダーテキスト（macOS情報＋選択行）をクリップボードへコピー
- **Shift + コピー**：macOS情報＋全アプリのバージョン一覧をまとめてコピー
- **更新ボタン**：再スキャン
- **フィルターメニュー**：アプリ名で絞り込み（日本語環境では全角括弧オプションあり）
- **右クリックメニュー**：選択行をコピー・Finderで表示
- **変更履歴ウィンドウ**
- ウィンドウ位置・サイズを保存・復元

## 動作環境

- macOS 13.0 Ventura 以降
- Apple Silicon / Intel 両対応

## ビルド

Xcode で `Aver.xcodeproj` を開いてビルドしてください。
